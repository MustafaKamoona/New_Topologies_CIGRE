# AC/DC Benchmarks

## Files and folders

- `acdc_7bus`: Simple benchmark with AC/DC network
    - `droop`: ini working, but run not
- `acdc_link`: CIGRE European LV network based AC/DC benchmark with end-of-feeder AC/DC link.
    - `droops`: ini working
- `acdc_6w`: CIGRE European LV network based AC/DC benchmark with 4 AC wires and 2 DC.
    - `droops`: ini working
- `acdc_4w`: CIGRE European LV network based AC/DC benchmark with 2 AC wires and 2 DC.
